## Agricultural Drones

UAV platforms designed specifically for agricultural tasks such as spraying, seeding, and monitoring.

- [DJI Agras Series](https://www.dji.com/agras) — Agricultural drones for spraying, spreading, and crop protection.
- [XAG Agricultural Drones](https://www.xa.com/en) — Autonomous drones for precision agriculture operations.
- [Yamaha RMAX](https://global.yamaha-motor.com/business/uav/) — Industrial UAV helicopter for crop spraying applications.

## Flight Control & Autonomy

Software and systems for controlling UAVs and enabling autonomous operations.

- [ArduPilot](https://ardupilot.org/) — Open-source autopilot system supporting agricultural UAV deployments.
- [PX4](https://px4.io/) — Open-source flight control software for drones and autonomous systems.
- [QGroundControl](https://docs.qgroundcontrol.com/) — Ground control station for mission planning and UAV management.

## Imaging & Sensors

Sensors and imaging technologies used for crop monitoring and field analysis.

- [MicaSense](https://www.micasense.com/) — Multispectral sensors for agricultural drone imaging.
- [Parrot Sequoia](https://www.parrot.com/en/drones/sequoia) — Multispectral sensor for crop analysis and vegetation mapping.
- [Sentera Sensors](https://sentera.com/) — Precision agriculture sensors for UAV-based imaging.

## Data Processing & Analytics

Platforms and tools for analyzing UAV-collected agricultural data.

- [DroneDeploy](https://www.dronedeploy.com/) — Aerial data platform for mapping, analysis, and insights.
- [Pix4Dfields](https://www.pix4d.com/product/pix4dfields) — Drone mapping software tailored for agriculture.
- [Agisoft Metashape](https://www.agisoft.com/) — Photogrammetry software for processing aerial imagery.

## Mapping & Surveying

Tools for creating maps, field models, and geospatial insights from UAV data.

- [OpenDroneMap](https://www.opendronemap.org/) — Open-source toolkit for processing aerial imagery into maps and models.
- [Mapbox](https://www.mapbox.com/) — Mapping platform for building geospatial applications.
- [QGIS](https://qgis.org/) — Open-source geographic information system for spatial analysis.

## Research & Open Source

Datasets, research initiatives, and open tools supporting UAV use in agriculture.

- [AgML](https://agml.ai/) — Open-source datasets and tools for agricultural machine learning.
- [Farm-ng](https://farm-ng.com/) — Open robotics and autonomy platform for agriculture.
- [Open Agriculture Initiative (MIT)](https://www.media.mit.edu/groups/open-agriculture-openag/overview/) — Research initiative focused on agriculture and technology.
